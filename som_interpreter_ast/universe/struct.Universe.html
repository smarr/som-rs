Expand description

The central data structure for the interpreter.

It represents the complete state of the interpreter, like the known class definitions, the string interner and the stack frames.

pub fn eval_block_with_frame_no_pop( &mut self, value_stack: &mut GlobalValueStack, nbr_locals: u8, nbr_args: usize, ) -> Return

Evaluates a block after pushing a new block frame, and the new frame doesn’t get created by popping arguments - just copying them. Implemented to help avoid an odd GC bug in the to:by:do node where a ref to a pointer, kept as a variable, ended up dangling after GC for some reason. This variable was popped off the stack and pushed back. Thanks to this function, it’s just copied from the previous one to the new one. Which may also be a speedup
