The time record of the interpreter’s creation.

§bytecode_idx: u16

The current bytecode index.

§current_frame: UnsafeCell<Gc<Frame>>

The current frame.

§current_bytecodes: *const Vec<Bytecode>

Pointer to the frame’s bytecodes, to not have to read them from the frame directly

§frame_method_root: Gc<Method>

GC can trigger when the interpreter wants to allocate a new frame. We’re then in a situation where we’ve looked up a Method (which is how we knew we were dealing with a non-primitive, and so that we had to create a frame) So this method can’t be stored on the Rust stack, or GC would miss it. Therefore: we keep it reachable there.
